# Copilot Studio Agent Building Toolkit

Prepare our partners for new specializations or audits can be time consuming and complex. To assist our partners, we are pleased to introduce SpecX AI Agent

# SpecX AI Agent – Declarative Build Guide

This repository helps partners build the **SpecX AI Agent** using
**Microsoft Copilot Studio (declarative agent only)**.

## What this repo is
- Step-by-step guidance to build ONE declarative agent
- Uses only instructions, knowledge, and starter prompts
- Designed for partner audit and specialization readiness scenarios

## What this repo is NOT
- No Power Automate flows
- No custom connectors
- No APIs or code extensions

## Quick Start
1. Review prerequisites
2. Create a new agent in Copilot Studio
3. Copy SpecX instructions
4. Connect knowledge sources
5. Add starter prompts
6. Test and publish
